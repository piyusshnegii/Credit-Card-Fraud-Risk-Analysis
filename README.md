# 💳 Credit Card Fraud Risk Analysis Dashboard

## 📌 Overview
This project analyzes credit card transaction data to identify fraud patterns, risk levels, and trends using Python and Power BI.

It provides an interactive dashboard to monitor fraudulent activities across different fraud types, states, and time periods.

---

## 🎯 Objectives
- Identify fraudulent transaction patterns
- Analyze risk levels (Low, Medium, High, Critical)
- Detect the most common fraud types
- Track fraud trends over time
- Compare fraud distribution across states

## 📈 Key Insights
- Fraud rate is **17.27%**, indicating significant risk exposure
- High-risk transactions contribute the largest share (**41.23%**)
- **Card Skimming** is the most expensive fraud type
- **Uttar Pradesh** has the highest fraud count
- Fraud activity shows noticeable peaks across certain months
